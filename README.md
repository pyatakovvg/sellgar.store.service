# sellgar.store.service

Сервис продаваемых товаров Sellgar.

Планируемая зона ответственности:

- `store_product` - продаваемый товар, связанный с catalog product;
- `store_variant` - продаваемый вариант, связанный с catalog variant;
- история цен продаваемых вариантов;
- остатки, резервы и будущие stock movements;
- публикация/доступность товара в конкретном магазине или канале.

Старую модель `store = one variant` не переносить механически. Схема сервиса должна проектироваться отдельно перед реализацией доменных таблиц.
